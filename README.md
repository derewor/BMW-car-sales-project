# BMW-car-sales-project
The project aimed to examine how the sale of BMW cars change over a decade period.

# Objective
To analyze the business performace of the BMW company for its different car products on global market. The output helps to design a business strategy to improve revenue.

# Tools used
Pandas, python sklearn tools, visualization toos (seaborn, matplotlib)
# Code Usage
The jupyter notebook in the src folder contains active code. With the data in the data folder can be run to reproduce the result.
